## Local Development

- Install

```
npm install
```

- Copy environment variables

```
cp .env.development.sample .env.development 
```

- Run

```
npm run dev
```

navigate to `http://localhost:5173/`

- Build

```
npm run build
```

- Docker compose

```
docker compose up
```

navigate to `http://localhost:8080/`
